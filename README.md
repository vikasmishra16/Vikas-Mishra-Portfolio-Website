## 📂 Project Structure

- `App.tsx`: The main entry point and UI logic.
- `constants.ts`: Portfolio data (experience, projects, skills, etc.).
- `components/`: UI components like `WarpGrid`.
- `public/`: Static assets including `resume.pdf`.

## 💻 Running Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
